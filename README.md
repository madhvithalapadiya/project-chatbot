# 🤖 Project Chatbot (Frontend)

This repository contains the frontend of a chatbot application developed by **Madhvi Thalapadiya**. The project focuses on creating an interactive and user-friendly chat interface.

---

## 📌 Project Overview

The **Project Chatbot** frontend is designed to provide a smooth and responsive UI for users to interact with a chatbot. It handles user input, displays messages, and ensures a seamless chatting experience.

---

## 🚀 Features

* 💬 Interactive chat interface
* ⚡ Real-time message display
* 🎨 Clean and responsive UI
* 🔄 Dynamic message rendering
* 📱 Mobile-friendly design

## ⚙️ Installation & Setup

1. Clone the repository:

   ```
   git clone https://github.com/madhvithalapadiya/project-chatbot.git
   ```

2. Navigate to the project folder:

   ```
   cd project-chatbot
   ```

3. Open `index.html` in your browser.
